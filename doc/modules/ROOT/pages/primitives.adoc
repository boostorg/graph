= Graph Interface Primitives

All BGL graph types share a common set of free functions for accessing
and modifying the graph structure. These functions are how algorithms
interact with graphs, regardless of the underlying data structure.

== Traversal

[cols="2,3,2"]
|===
| Function | Returns | Concept

| `vertices(g)`
| Pair of vertex iterators (all vertices)
| xref:concepts/VertexListGraph.adoc[VertexListGraph]

| `edges(g)`
| Pair of edge iterators (all edges)
| xref:concepts/EdgeListGraph.adoc[EdgeListGraph]

| `out_edges(v, g)`
| Pair of out-edge iterators
| xref:concepts/IncidenceGraph.adoc[IncidenceGraph]

| `in_edges(v, g)`
| Pair of in-edge iterators
| xref:concepts/BidirectionalGraph.adoc[BidirectionalGraph]

| `adjacent_vertices(v, g)`
| Pair of adjacency iterators
| xref:concepts/AdjacencyGraph.adoc[AdjacencyGraph]
|===

NOTE: All pairs can be used with `make_iterator_range` for range-based for loops (see example above).

== Mutation

[cols="2,3,2"]
|===
| Function | Effect | Concept

| `add_vertex(g)`
| Add a vertex, return its descriptor
| xref:concepts/MutableGraph.adoc[MutableGraph]

| `remove_vertex(v, g)`
| Remove vertex `v` and all incident edges
| xref:concepts/MutableGraph.adoc[MutableGraph]

| `add_edge(u, v, g)`
| Add edge from `u` to `v`, return pair (descriptor, bool)
| xref:concepts/MutableGraph.adoc[MutableGraph]

| `remove_edge(u, v, g)`
| Remove edge(s) between `u` and `v`
| xref:concepts/MutableGraph.adoc[MutableGraph]

| `remove_edge(e, g)`
| Remove specific edge `e`
| xref:concepts/MutableGraph.adoc[MutableGraph]

| `clear_vertex(v, g)`
| Remove all edges incident to `v` (vertex remains)
| xref:concepts/MutableGraph.adoc[MutableGraph]
|===

WARNING: With `adjacency_list<vecS, vecS, ...>`, `remove_vertex()`
invalidates all vertex descriptors after the removed vertex. See
xref:graph_classes/overview.adoc[Graph Data Structures] for details.

== Property access

[cols="2,3"]
|===
| Function | Returns

| `get(&Struct::member, g)`
| Property map for bundled property member

| `get(map, key)`
| Value of `key` in property map `map`

| `put(map, key, value)`
| Set value of `key` in property map `map`

| `g[v]` or `g[e]`
| Direct reference to bundled property struct of vertex `v` or edge `e`

| `get(tag, g)`
| Property map for internal property tag (legacy, prefer bundled properties)

== Types

Obtained via `graph_traits<Graph>`:

[cols="1,3"]
|===
| Type | Description

| `vertex_descriptor`
| Handle to a vertex. Opaque type, do not assume it is an integer.

| `edge_descriptor`
| Handle to an edge. Contains source and target information.

| `vertex_iterator`
| Iterator over all vertices.

| `edge_iterator`
| Iterator over all edges.

| `out_edge_iterator`
| Iterator over the out-edges of a vertex.

| `in_edge_iterator`
| Iterator over the in-edges of a vertex (requires `bidirectionalS`).

| `adjacency_iterator`
| Iterator over the adjacent vertices of a vertex.

| `vertices_size_type`
| Unsigned integer type for vertex counts.

| `edges_size_type`
| Unsigned integer type for edge counts.

| `degree_size_type`
| Unsigned integer type for vertex degree.
|===
