= Modern Examples

Self-contained examples using modern C++ idioms and bundled properties.

== Dijkstra with Bundled Properties

Shortest paths on a small directed graph using a user-defined `Road` struct
as edge property and `get(&Road::cost, g)` to create the weight map.

[source,cpp]
----
include::example$dijkstra_bundled.cpp[]
----

Expected output:

----
distance to vertex 0 = 0
distance to vertex 1 = 1
distance to vertex 2 = 3
distance to vertex 3 = 4
----
