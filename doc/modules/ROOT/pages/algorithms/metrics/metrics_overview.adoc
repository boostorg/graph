= Graph Metrics

Quantitative measures that describe the structure of a graph or the role of
individual vertices and edges within it.

== Centrality

How important is a vertex?

[cols="1,1,3"]
|===
| Metric | Complexity | What it measures

| xref:algorithms/metrics/betweenness_centrality.adoc[Betweenness Centrality]
| O(V * E)
| How often a vertex lies on shortest paths between other vertices.
  High betweenness = bridge or bottleneck.

| Closeness Centrality
| O(V * (V + E))
| Inverse of the average distance to all other vertices.
  High closeness = can reach everyone quickly.

| Degree Centrality
| O(1) per vertex
| Number of connections. Simple but effective for many networks.

| PageRank
| O(V + E) per iteration
| Importance based on incoming links (recursive: a link from an
  important page counts more). Used by web search engines.
|===

== Distance and structure

[cols="1,3"]
|===
| Metric | What it measures

| Eccentricity
| Maximum distance from a vertex to any other. The minimum eccentricity
  across all vertices is the graph's _radius_; the maximum is the _diameter_.

| Geodesic Distance
| Mean shortest-path distance from a vertex. Low mean = centrally located.

| Clustering Coefficient
| Fraction of a vertex's neighbors that are also neighbors of each other.
  High clustering = tight-knit local community.

| xref:algorithms/metrics/profile.adoc[Profile] / xref:algorithms/metrics/wavefront.adoc[Wavefront] / xref:algorithms/metrics/bandwidth.adoc[Bandwidth]
| Sparse matrix metrics used to evaluate vertex orderings for numerical solvers.
|===

== Graph-level statistics

[cols="1,3"]
|===
| Metric | What it measures

| Core Numbers
| k-core decomposition: assigns each vertex the largest k such that it
  belongs to a subgraph where every vertex has degree >= k. Useful for
  identifying dense substructures.
|===
