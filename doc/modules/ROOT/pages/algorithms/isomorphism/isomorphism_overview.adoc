= Graph Isomorphism

Test whether two graphs have the same structure (a bijection between vertices
that preserves edges).

[cols="1,1,3"]
|===
| Algorithm | Complexity | When to use

| xref:algorithms/isomorphism/isomorphism.adoc[Isomorphism]
| O(V! * V) worst case
| Test if two graphs are isomorphic (same structure, same size).

| xref:algorithms/isomorphism/vf2_sub_graph_iso.adoc[VF2 Subgraph Isomorphism]
| O(V! * V) worst case
| Test if one graph is isomorphic to a subgraph of another.
  Finds all matches via a callback.

| xref:algorithms/isomorphism/mcgregor_common_subgraphs.adoc[McGregor Common Subgraphs]
| Exponential
| Find the largest subgraph common to two graphs.
|===

NOTE: All graph isomorphism algorithms have exponential worst-case complexity.
In practice they are fast on most real-world graphs due to pruning, but they
can be slow on highly symmetric or regular graphs.
