= DFS Visitor Adaptor

Converts a list of event visitors into a DFSVisitor for use with
`depth_first_search`.

*Defined in:* `<boost/graph/depth_first_search.hpp>` +
*Models:* DFSVisitor

== Example

[source,cpp]
----
include::example$visitors/dfs_visitor.cpp[]
----

[,text]
----
include::example$visitors/dfs_visitor.txt[]
----

== Synopsis

[source,cpp]
----
template <typename EventVisitorList = null_visitor>
class dfs_visitor;

template <typename EventVisitorList>
dfs_visitor<EventVisitorList>
make_dfs_visitor(EventVisitorList ev_list);
----

== Parameters

[cols="1,3,1",options="header"]
|===
| Parameter | Description | Default

| `EventVisitorList`
| One or more EventVisitors combined with `std::pair`.
| `null_visitor`
|===

== Behavior

Implements all DFSVisitor event points. Each event is dispatched to the
matching EventVisitor in the list. Events with no matching visitor are ignored.
