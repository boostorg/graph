= astar_visitor

....

astar_visitor<EventVisitorList>
....

This class is an adapter that converts a list of
xref:visitors/EventVisitor.adoc[EventVisitor]'s (constructed using `std::pair`)
into a xref:visitors/AStarVisitor.adoc[AStarVisitor].

==== Example

See the example for xref:visitors/EventVisitor.adoc[EventVisitor].

==== Model of

xref:visitors/AStarVisitor.adoc[AStarVisitor]

==== Template Parameters

[cols=",,",options="header",]
|===
|Parameter |Description |Default
|`EventVisitorList` |A list of xref:visitors/EventVisitor.adoc[EventVisitor]'s
created with `std::pair`. |xref:visitors/null_visitor.adoc[`null_visitor`]
|===

==== Where Defined

link:../../../boost/graph/astar_search.hpp[`boost/graph/astar_search.hpp`]

==== Member Functions

This class implements all of the member functions required by
xref:visitors/AStarVisitor.adoc[AStarVisitor]. In each function the appropriate
event is dispatched to the xref:visitors/EventVisitor.adoc[EventVisitor]'s in
the EventVisitorList.

==== Non-Member Functions

[width="100%",cols="50%,50%",options="header",]
|===
|Function |Description
|` template <class EventVisitorList>` +
`astar_visitor<EventVisitorList>` +
`make_astar_visitor(EventVisitorList ev_list); ` |Returns
the event visitor list adapted to be an A* visitor.
|===
